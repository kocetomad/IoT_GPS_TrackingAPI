How to use the managment tab?
The management tab is where devices are "added" to the user's profile. If a device is succesfully added it will be immediately visible in the "Locate devices" tab. Whenever a new device is added the user must specify a unique device Id number and a "label" message.
What is a device ID?
The Id of a device is a uniquely assigned number that should be provided with each newly bought device. Each and every device part of the system, has an ID number. From the moment a device gets powered on for the first time it will immidetly send it's location to the central server but in order for a user to claim a device as their own they have to "register" the id in order for the device to become "visible" for them.
What is a device Label?
Labels are used as a method of destinguishing devices by somethong more than just a plain device id. Labels are not mandatory but greatly improve the general user experience. A label can be as long as the user wishesh(255 charecters at most) but it strongly recomended to use a short and descriptive one rather than a long and convaluted label.
